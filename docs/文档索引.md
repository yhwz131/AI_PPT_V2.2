## 前端目录对照（易混）

| 目录 | 角色 |
|------|------|
| **`frontend-new/`** | 当前主用的 Vue 3 + Vite + TypeScript 工程；`setup_env.sh` / 注释中构建产物拷贝到 **`frontend/`** |
| **`frontend/`** | Nginx 静态根目录（构建产物），非源码主目录 |
| **`new_frontend/`** | 另一套前端（如 Element Plus），**不是** `setup_env.sh` 默认部署源 |

## 文档文件名前缀「jna」

部分架构/数据流文档以 `jna` 开头，仓库内**无单独全称定义**，可作内部技术文档系列前缀；详见 [各服务配置与常见问题说明.md](各服务配置与常见问题说明.md) 文首说明。
